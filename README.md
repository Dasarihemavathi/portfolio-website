# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, projects, technical expertise, and professional journey as a Python Full Stack Developer.

## 🚀 About the Project

This is a fully responsive portfolio website designed to provide a professional online presence. It highlights my technical skills, featured projects, education, certifications, and contact information in a clean and modern interface.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and user-friendly interface
- About Me section
- Technical Skills section
- Featured Projects
- Education & Certifications
- Contact Information
- Smooth navigation
- Clean and maintainable code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 💻 Getting Started

Clone the repository:

```bash
git clone https://github.com/Dasarihemavathi/portfolio-website.git
```

Navigate to the project folder:

```bash
cd portfolio-website
```

Open `index.html` in your browser.
