# Dasari Hemavathi Portfolio

Personal portfolio website for Python full stack developer opportunities.

## Tech Used

- HTML for page structure
- CSS for responsive design and styling
- JavaScript for mobile navigation, scroll reveal, and active section highlighting
- Python for running the local development server

## Main Files

- `index.html` - portfolio content
- `portfolio.css` - complete visual design
- `portfolio.js` - interactive behavior
- `server.py` - local Python server
- `assets-profile-headshot.jpeg` - profile photo

## Run Locally

```bash
npm run dev
```

Then open:

```text
http://127.0.0.1:8000
```

You can also open `index.html` directly in the browser.

## GitHub Push Steps

```bash
git add .
git commit -m "Create professional portfolio website"
git branch -M main
git remote add origin https://github.com/Dasarihemavathi/YOUR_REPOSITORY_NAME.git
git push -u origin main
```

Replace `YOUR_REPOSITORY_NAME` with your GitHub repository name.
