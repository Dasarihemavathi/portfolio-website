# Project Recommendation System

A rule-based recommendation system that suggests project ideas from a user's skills, interests, preferred domain, and difficulty level.

## Features

- Collects user skills, interests, domain, and difficulty
- Scores project ideas with transparent rule-based logic
- Shows personalized project recommendations in an interactive UI
- Includes a JSON API for recommendation results

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript

## Run Locally

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver 5002
```

Then open:

```text
http://127.0.0.1:5002
```
