# Simple Student Management System

A lightweight browser-based student management system for adding, editing, deleting, and searching student records.

## Features

- Add student records with name, roll number, course, grade, and email
- Edit existing student details
- Delete student records
- Search students by name, roll number, or course
- Dashboard totals for students, average grade, and active courses
- Saves records in the browser with `localStorage`

## Run

Open `index.html` in any modern browser.

No build step or server is required.
